# houston2026-concierge

Single-page landing site for **Houston 2026 Concierge** — boutique trip planning for the FIFA World Cup 2026.

**Live site:** https://houston2026-concierge.pages.dev/

## What's in here

- `index.html` — the landing page (single file, embedded CSS, no build step)
- `images/` — hero, services, watch-party visuals (AI-generated via Pollinations/Gemini)
- `copy.json` — source copy generated via rig llama-3.3-70b-versatile

## Deploy

Hosted on Cloudflare Pages. To redeploy after edits:

```bash
npx wrangler pages deploy . --project-name=houston2026-concierge
```
